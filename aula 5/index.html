Principais formatações

Negrito / Destaque

Para deixar um texto em negrito basta usar a tag B (não semântica)

Para deixar um texto em destaque basta usar a tag STRONG (semântica)

Itálico / Ênfase

Para deixar um texto em itálico basta usar a tag I (não semântica)

para deixar um texto em ênfase basta usar a tag EM (semântico)

Texto marcado

Para deixar um texto marcado basta usar a tag MARK

Texto grande e pequeno

Para deixar um texo grande basta usar a tag BIG (Entrando em desuso) e para deixar um texto pequeno basta usar a tag SMALL

Texto excluido

Para deixar um texto em exclusão basta usar a tag DEL

Texto inserido / sublinhado

Para deixar um texto em inserção basta usar a tag INS (semântica)

Para deixar um texto sublinhado basta usar a tag U (não semântica)

Texto sobrescrito

Exemplo: x3+5x2+3=0 usa-se a tag SUP

Texto subscrito

Exemplo: H2O usa-se a tag SUB
